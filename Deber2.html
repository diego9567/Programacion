DEBER - BUCLES

Estos ejercicios utilizan estructuras repetitivas (for y while) en Python para repetir acciones y trabajar con secuencias de datos.

Ejercicio 1 - Números del 1 al 10

for i in range(1, 11):
    print(i)

Ejercicio 2 - Números pares del 1 al 20

for i in range(1, 21):
    if i % 2 == 0:
        print(i)

Ejercicio 3 - Tabla de multiplicar

numero = int(input("Ingrese un número: "))

for i in range(1, 11):
    print(numero, "x", i, "=", numero * i)

Ejercicio 4 - Números del 10 al 1

for i in range(10, 0, -1):
    print(i)

Ejercicio 5 - Suma acumulada

numero = int(input("Ingrese un número: "))
suma = 0

for i in range(1, numero + 1):
    suma += i

print("La suma es:", suma)

Ejercicio 6 - Contar vocales

palabra = input("Ingrese una palabra: ")
contador = 0

for letra in palabra:
    if letra.lower() in "aeiou":
        contador += 1

print("Cantidad de vocales:", contador)

Ejercicio 7 - Contraseña correcta

clave_correcta = "python123"
clave = input("Ingrese la contraseña: ")

while clave != clave_correcta:
    print("Contraseña incorrecta")
    clave = input("Intente nuevamente: ")

print("Acceso permitido")

Ejercicio 8 - Múltiplos de 5

for i in range(1, 51):
    if i % 5 == 0:
        print(i)

Ejercicio 9 - Mayor de 5 números

mayor = int(input("Ingrese un número: "))

for i in range(4):
    numero = int(input("Ingrese un número: "))
    if numero > mayor:
        mayor = numero

print("El número mayor es:", mayor)

Ejercicio 10 - Triángulo de asteriscos

filas = 5

for i in range(1, filas + 1):
    print("*" * i)
