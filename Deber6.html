DEBER - TKINTER (NOTAS)

Este programa está desarrollado en Python utilizando la librería Tkinter para crear una interfaz gráfica. Permite ingresar varias notas, almacenarlas en una lista y calcular el promedio. El programa indica si el estudiante aprueba o reprueba según el promedio obtenido.

Código del programa

import tkinter as tk

notas = []

def agregar_nota():
    try:
        nota = float(entry.get())
        notas.append(nota)
        lista_label.config(text=f"Notas: {notas}")
        entry.delete(0, "end")   
    except:
        resultado.config(text="Ingrese un número válido")

def calcular_promedio():
    if len(notas) > 0:
        promedio = sum(notas) / len(notas)
        
        if promedio >= 7:
            estado = "Aprueba"
        else:
            estado = "Reprueba"
        
        resultado.config(text=f"Promedio: {round(promedio,2)}\n{estado}")
    else:
        resultado.config(text="No hay notas")

ventana = tk.Tk()
ventana.geometry("250x300")
ventana.title("Notas")

frame = tk.Frame(ventana, bg="yellow")
frame.pack(fill="both", expand=True)

label = tk.Label(frame, text="Ingrese sus notas",
font=("Arial",12,"bold"), bg="yellow", fg="blue")
label.pack()

entry = tk.Entry(frame)
entry.pack()

btn_agregar = tk.Button(frame, text="Agregar", command=agregar_nota)
btn_agregar.pack()

btn_promedio = tk.Button(frame, text="Promedio", command=calcular_promedio)
btn_promedio.pack()

lista_label = tk.Label(frame, text="Notas: []", bg="yellow")
lista_label.pack()

resultado = tk.Label(frame, text="", bg="yellow")
resultado.pack()

ventana.mainloop()
