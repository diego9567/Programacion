DEBER - MENÚ DE OPCIONES (CALCULADORA)

Este programa en Python implementa una calculadora básica utilizando un menú de opciones. El usuario puede elegir entre sumar, restar, multiplicar o dividir. El programa solicita dos números y realiza la operación seleccionada mediante funciones.

Código del programa

def menu():
    print("Calculadora básica")
    print("1. Sumar")
    print("2. Restar")
    print("3. Multiplicar")
    print("4. Dividir")

    opcion = pedirNumero("Ingrese una opción: ")

    if opcion == 1:
        num1 = pedirNumero("Ingrese el primer número: ")
        num2 = pedirNumero("Ingrese el segundo número: ")
        sumar(num1, num2)

    elif opcion == 2:
        num1 = pedirNumero("Ingrese el primer número: ")
        num2 = pedirNumero("Ingrese el segundo número: ")
        restar(num1, num2)

    elif opcion == 3:
        num1 = pedirNumero("Ingrese el primer número: ")
        num2 = pedirNumero("Ingrese el segundo número: ")
        multiplicar(num1, num2)

    elif opcion == 4:
        num1 = pedirNumero("Ingrese el primer número: ")
        num2 = pedirNumero("Ingrese el segundo número: ")
        dividir(num1, num2)

    else:
        print("Opción no válida")


def pedirNumero(mensaje):
    return int(input(mensaje))


def sumar(num1, num2):
    print("La suma es:", num1 + num2)


def restar(num1, num2):
    print("La resta es:", num1 - num2)


def multiplicar(num1, num2):
    print("La multiplicación es:", num1 * num2)


def dividir(num1, num2):
    if num2 != 0:
        print("La división es:", num1 / num2)
    else:
        print("No se puede dividir para cero")


menu()
