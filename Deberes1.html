DEBER - SENTENCIA IF

Este conjunto de ejercicios utiliza estructuras condicionales (if) en Python para tomar decisiones según los datos ingresados por el usuario.

Ejercicio 1 - Número mayor que 50

numero = int(input("Ingrese un número: "))

if numero > 50:
    print("El número es mayor que 50")
else:
    print("El número no es mayor que 50")

Ejercicio 2 - Puede votar

edad = int(input("Ingrese su edad: "))

if edad >= 18:
    print("Puede votar")
else:
    print("No puede votar")

Ejercicio 3 - Múltiplo de 3

numero = int(input("Ingrese un número: "))

if numero % 3 == 0:
    print("Es múltiplo de 3")
else:
    print("No es múltiplo de 3")

Ejercicio 4 - Contraseña

clave = input("Ingrese la contraseña: ")

if clave == "python123":
    print("Contraseña correcta")
else:
    print("Contraseña incorrecta")

Ejercicio 5 - Aprobado o reprobado

nota = float(input("Ingrese la nota (0-10): "))

if nota >= 7:
    print("Aprobado")
else:
    print("Reprobado")

Ejercicio 6 - Número mayor entre dos

num1 = int(input("Ingrese el primer número: "))
num2 = int(input("Ingrese el segundo número: "))

if num1 > num2:
    print("El mayor es:", num1)
elif num2 > num1:
    print("El mayor es:", num2)
else:
    print("Ambos números son iguales")

Ejercicio 7 - Tipo de número

numero = int(input("Ingrese un número: "))

if numero > 0:
    print("Es positivo")
elif numero < 0:
    print("Es negativo")
else:
    print("Es cero")

Ejercicio 8 - Vocal o no

letra = input("Ingrese una letra: ")

if letra.lower() in "aeiou":
    print("Es una vocal")
else:
    print("No es una vocal")

Ejercicio 9 - Temperatura

temperatura = float(input("Ingrese la temperatura en °C: "))

if temperatura < 15:
    print("Hace frío")
else:
    print("No hace frío")

Ejercicio 10 - Año bisiesto

anio = int(input("Ingrese un año: "))

if (anio % 4 == 0 and anio % 100 != 0) or (anio % 400 == 0):
    print("Es un año bisiesto")
else:
    print("No es un año bisiesto")
