DEBERES SENTENCIA IF

Este ejercicio trata sobre el uso de condicionales (if) en Python para tomar decisiones según los datos que ingresa el usuario. En el primer caso, el programa solicita un número y verifica si es mayor que 50 para mostrar un mensaje correspondiente. En el segundo caso, el programa pide la edad de una persona y determina si puede votar, considerando que solo pueden hacerlo las personas que tienen 18 años o más.

Código Ejercicio 1

numero = int(input("Ingrese un número: "))

if numero > 50:
    print("El número es mayor que 50")
else:
    print("El número no es mayor que 50")

Código Ejercicio 2

nombre = input("Ingrese su nombre: ")
edad = int(input("Ingrese su edad: "))

if edad >= 18:
    print(nombre, "puede votar")
else:
    print(nombre, "no puede votar")
