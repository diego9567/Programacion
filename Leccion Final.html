LECCIÓN FINAL - PROYECTO DE INVENTARIO

En esta sección se explica la teoría utilizada en el desarrollo del proyecto de inventario con interfaz gráfica en Python.

📌 Variables

Las variables son espacios en memoria donde se almacenan datos que pueden cambiar durante la ejecución del programa. Se utilizan para guardar información como el nombre del producto y la cantidad. 

producto = "sal"
cantidad = 5

📌 Diccionarios

Un diccionario es una estructura de datos que almacena información en pares clave-valor. Se utiliza para guardar el inventario. 

inventario = {"sal": 5}

📌 Funciones

Las funciones son bloques de código reutilizables que realizan tareas específicas. 

def agregar():
    print("Agregando producto")

📌 Condicionales

Permiten tomar decisiones en el programa dependiendo de una condición. 

if producto in inventario:
    print("El producto ya existe")

📌 Estructuras de repetición (for)

Permiten repetir un bloque de código varias veces. 

for producto, cantidad in inventario.items():
    print(producto, cantidad)

📌 Entrada de datos

Permite que el usuario ingrese información al programa mediante campos de texto. 

producto = entry_producto.get()

📌 Interfaz gráfica (Tkinter)

Permite interactuar con el programa mediante ventanas, botones y campos de texto. 

ventana = tk.Tk()

📌 Eventos

Son acciones que ocurren cuando el usuario interactúa con la interfaz. 

tk.Button(ventana, text="Agregar", command=agregar)

📌 Actualización de interfaz

Permite actualizar la información mostrada en pantalla en tiempo real. 

label_lista.config(text="Inventario actualizado")

Conclusión

En este proyecto se aplicaron conceptos fundamentales como variables, funciones, condicionales, diccionarios y estructuras de repetición, junto con una interfaz gráfica en Tkinter. Esto permitió desarrollar una aplicación funcional para el control de inventario.
