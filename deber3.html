DEBER - TKINTER

Este programa está hecho en Python utilizando la librería Tkinter, que permite crear interfaces gráficas. El programa abre una ventana donde el usuario puede ingresar su nombre y su edad en dos cajas de texto. Luego, al presionar el botón “Saludar”, se ejecuta una función que obtiene los datos escritos por el usuario y muestra en la terminal un mensaje con el saludo, indicando el nombre y la edad ingresados.

Código Ejercicio 1

import tkinter as tk

def saludar():
    nombre = caja_nombre.get()
    edad = caja_edad.get()
    print("Hola " + nombre + " tienes " + edad + " años")

ventana = tk.Tk()
ventana.geometry("250x200")
ventana.title("Saludo")

texto1 = tk.Label(ventana, text="Ingrese su nombre")
texto1.pack()

caja_nombre = tk.Entry(ventana)
caja_nombre.pack()

texto2 = tk.Label(ventana, text="Ingrese su edad")
texto2.pack()

caja_edad = tk.Entry(ventana)
caja_edad.pack()

button = tk.Button(ventana, text="Saludar", command=saludar)
button.pack()

ventana.mainloop()
