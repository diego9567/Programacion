DEBER - FUNCIONES EN PYTHON

Estos ejercicios utilizan funciones en Python para organizar el código, reutilizar lógica y devolver resultados mediante el uso de parámetros y valores de retorno.

Ejercicio 1 - Mensaje de bienvenida

def mostrar_bienvenida(nombre):
    print("Bienvenido,", nombre)

Ejercicio 2 - Par o impar

def es_par(numero):
    if numero % 2 == 0:
        return "Par"
    else:
        return "Impar"

Ejercicio 3 - Suma de dos números

def sumar(num1, num2):
    return num1 + num2

Ejercicio 4 - Longitud de una palabra

def longitud_palabra(palabra):
    return len(palabra)

Ejercicio 5 - Cuadrado de un número

def cuadrado(numero):
    return numero ** 2

Ejercicio 6 - Mayor de tres números

def mayor_de_tres(a, b, c):
    if a >= b and a >= c:
        return a
    elif b >= a and b >= c:
        return b
    else:
        return c

Ejercicio 7 - Celsius a Fahrenheit

def celsius_a_fahrenheit(celsius):
    return (celsius * 9/5) + 32

Ejercicio 8 - Suma de una lista

def suma_lista(lista):
    suma = 0
    for numero in lista:
        suma += numero
    return suma

Ejercicio 9 - Número positivo

def es_positivo(numero):
    if numero > 0:
        return True
    else:
        return False

Ejercicio 10 - Invertir palabra

def invertir_palabra(palabra):
    return palabra[::-1]
