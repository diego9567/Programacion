LECCIÓN FINAL TEORICO - MINI INVENTARIO

Este programa está desarrollado en Python utilizando Tkinter. Permite ingresar un producto mediante una interfaz gráfica y mostrar un mensaje cuando se agrega correctamente. También valida que el campo no esté vacío.

Código del programa

import tkinter as tk
from tkinter import messagebox

def agregar_prodcuto():
    producto = entry_producto.get()

    if producto == "":
        messagebox.showerror("Error", "Ingrese un producto")
    else:
        label_resultado.config(text=f"Producto agregado: {producto}")

ventana = tk.Tk()
ventana.title("Mini inventario")
ventana.geometry("250x200")

tk.Label(ventana, text="Ingrese un producto").pack()

entry_producto = tk.Entry(ventana)
entry_producto.pack()

tk.Button(ventana, text="Agregar", command=agregar_prodcuto).pack()

label_resultado = tk.Label(ventana, text="")
label_resultado.pack()

ventana.mainloop()
